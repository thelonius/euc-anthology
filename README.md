# Моноколесо Изнутри

Интерактивная антология реверс-инжиниринга прошивки электрического моноколеса. На примере **Begode ET Max** (STM32F405, 168V, FOC).

**Онлайн:** https://thelonius.github.io/euc-anthology/

## О чём это

14 глав с разбором того, как работает прошивка EUC: от физики балансировки и IMU до field weakening и тепловой модели обмоток. Каждая глава содержит:

- Прозаическое объяснение концепции
- Фрагменты реального декомпилированного кода с адресами в прошивке
- Интерактивную симуляцию (Canvas/ECharts), иллюстрирующую алгоритм
- Глоссарий терминов с тултипами

## Структура

```
Пролог · Падение                   — почему моноколесо едет
Часть I · Тело
  I.   Железо                      — STM32, TIM, ADC, блок-схема
  II.  Чувства                     — MPU6500, комплементарный фильтр
  III. Пробуждение                 — boot sequence, GYRO_BIAS калибровка
Часть II · Архитектура
  IV.  Состояния                   — конечный автомат прошивки
  V.   Цикл                        — хронометраж 50-мкс FOC ISR
  VI.  Карта                       — motor_state SRAM-структура
Часть III · Управление
  VII.  Равновесие                 — PD-регулятор баланса
  VIII. Поток                      — FOC, Clarke/Park, полный симулятор
  IX.   Наблюдатель                — Flux Observer vs Hall
  X.    Ослабление                 — field weakening, D-Q плоскость
Часть IV · Границы
  XI.   Тепло                      — двухмассовая модель обмоток
  XII.  Голос                      — BLE-телеметрия, протокол 0x55AA
Часть V · Практика
  XIII. Ремесло                    — SWD, дамп прошивки, Ghidra
```

## Технологии

- **Preact** — лёгкая альтернатива React
- **Vite** — сборка и dev-сервер
- **ECharts** — графики фазных токов и D-Q вектора
- **Canvas 2D** — кастомные визуализации (маятник, тепло, timeline)

Без бэкенда, без сборки CSS-фреймворка. Всё в одном бандле ~1.35 МБ (gzip 440 КБ).

## Локальный запуск

```bash
npm install
npm run dev       # dev-сервер на http://localhost:5173
npm run build     # production-сборка в dist/
npm run preview   # проверить production-сборку локально
```

## Деплой

Автоматический через GitHub Actions при push в `main`:

- `.github/workflows/deploy.yml` собирает сайт
- Артефакт заливается на GitHub Pages
- URL: `https://thelonius.github.io/euc-anthology/`

Чтобы включить Pages: **Settings → Pages → Source: GitHub Actions**.

## Источники

- Прошивка ET Max разобрана в Ghidra с нуля, адреса в коде указывают на реальные функции в образе
- Математика FOC — классические материалы по PMSM drives
- Параметры мотора (Kv, Rs, Ld, Lq, pole pairs) — из сервисной документации Begode и измерений сообщества
